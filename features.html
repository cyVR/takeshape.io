Developer

Multiple Environments

Seperate environments for development, staging and production. Move seamlessly from one to the other with automatic migrations.

Multiple Projects

One login for all of your TakeShape projects.

API

Supports both high-throughput read and writes. REST and GraphQL endpoints.

Hosting

We host your site with options for: custom domains. HTTPS, case-insensitive paths, URL redirects, and atomic deploys.

Internationalization (i18n)

Add support for any language you wish.

Search

Faceted search, geo search, full text search

CLI

Use the ```ts``` command line tool to create and deploy projects

Static Site Generator

Built-in static site generator with a flexible asset pipeline, local image proxy, and easy-to-use templating language (Nunjucks).

Tech Stack

TakeShape is built on rock solid technologies to keep you up and running for the long term. We run on an AWS serverless architecture. We use triple-redundant DynamoDB and S3 to store your content and assets. We want your content to be available lickey quick so we front everything with a CDN. Our current unit test code coverage is 65% and rising.
