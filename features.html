Tech Stack

TakeShape is built on rock solid technologies to keep you up and running for the long term. We run on an AWS serverless architecture. We use triple-redundant DynamoDB and S3 to store your content and assets. We want your content to be available lickey quick so we front everything with a CDN. Our current unit test code coverage is 65% and rising.
